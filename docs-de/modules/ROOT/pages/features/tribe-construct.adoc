Unter dem Menüpunkt Stammeskonstrukt kann ein Stammeskonstrukt erstellt und verwaltet werden. Das Stammeskonstrukt vereint die gemeinsame Verwendung von TW Forge durch mehrere Accounts:

- Hochgeladenen Berichte und eingehende Angriffe werden miteinander geteilt
- Abfragen für Aktionen zur Berücksichtigung der Abschickzeiten etc. der individuellen Accounts
- Teilen der Angriffspläne von berechneten Aktionen
- Verwaltung von Bündnissen zu anderen Stammeskonstrukten
- etc...
