Auf dieser Seite werden alle verfügbaren weltenübergreifenden Einstellungen in TW Forge erklärt.
Einstellungen beeinflussen die Anzeige von ausgewerteten Angriffen, die Darstellung im Interface, Discord-Verbindungen und Notifications.

== Allgemein

// hier ggf. nen Screenshot von den allgemeinen Settings (atm noch unnötig weil zu wenig Einstellmöglichkeiten

Unter diesem Punkt können grundlegende Anzeigeeinstellungen der Applikation vorgenommen werden.

Hier lässt sich das Erscheinungsbild von TW Forge festlegen. Zur Auswahl stehen:

- *Light Mode*
- *Dark Mode*
- *Browser-Standard*

Die gewählte Einstellung wirkt sich ausschließlich auf die Darstellung der Benutzeroberfläche aus und hat keinen Einfluss auf Funktionen, Berechnungen oder gespeicherte Daten.

== Angriffe

image::settings/angriffe_settings.png[Einstellung der farblichen Markierungen für eingehende Angriffe]

Unter diesem Punkt kann definiert werden, ab welchen maximal errechneten Truppen TW-Forge die farblichen Markierungen für die Incs vornimmt. Diese Einstellbarkeit sorgt dafür, dass individuelle Präferenzen umgesetzt werden können. 

"Kleine Angriffe" können durch den Regler angepasst werden, diese verschiebt dann wiederum die Untergrenze für klein-mittlere Angriffe, der Regler der klein-mittleren Angriffe verschiebt wiederum die eigene Obergrenze sowie die Untergrenze für mittlere Incs usw. 

Die Einstellungen, welche Farbe wann angezeigt wird, hat sowohl Auswirkungen auf die Anzeige auf TW-Forge unter xref:features/analyser.adoc#filterung[Angriffe] als auch auf die Anzeige Ingame durch das erlaubte Userscript.

== Discord

[cols="1,1", frame=none, grid=none]
|===
a| image::settings/discord_no_connection_settings.png[Keine Discordverknüpfung, width=100%]
a| image::settings/discord_connected_account.png[Discord verknüpft, width=100%]
|===

Unter diesem Punkt kann zunächst ein Discord Account verknüpft werden. Hierzu einfach auf den Button drücken und im Browser seinen Discord Account anmelden und der Verknüpfung zustimmen. So sind die Accounts erfolgreich verknüpft.

Jetzt hat man die Möglichkeit, entweder die Verknüpfung wieder zu löschen oder den Discord Bot zu einem Discord-Server (entweder Stammesdiscord oder Accountdiscord) hinzuzufügen.
 Außerdem besteht hier die Möglichkeit, dem Community-Discord "TW-Forge" zu joinen, um Fragen zu stellen sowie Feedback einzubringen.
